{{% steps %}}

### Шаг 1
Левой рукой возьмитесь за рукоятку нижнего блока кроссовера, положите руку трицепсом на подушку, для изолированного сгибания рук (скамье Скотта). Согните руку так, чтобы рукоятка была на уровне плеча. Это исходное положение.

### Шаг 2
На вдохе, медленно выпрямляйте руку до полного растяжения бицепса.

### Шаг 3
На выдохе, напрягая бицепс, сгибайте руку к плечу. Для максимального напряжения мышцы, разворачивайте кисть по часовой стрелке (мизинец должен оказаться выше большого пальца).

### Шаг 4
Выполните требуемое количество повторов, после чего поменяйте руку и сделайте то же самое.

{{% /steps %}}

{{< callout type="info" >}}
Не используйте силу инерции, выполняйте медленное контролируемое движение по полной амплитуде.
﻿﻿Подмышечная впадина упирается в угол скамьи Скотта. 
Следите за дыханием.
{{< /callout >}}
